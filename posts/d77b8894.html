Python pass

原文：https://www.javatpoint.com/python-pass

在 Python 中，pass关键字用于不执行任何内容；这意味着，当我们不想执行代码时，可以使用 pass 来执行 empty。它和名字所指的一样。它只是让控件在不执行任何代码的情况下通过。如果我们想绕过任何代码，可以使用 pass语句。

当一条语句在语法上是必需的时，这是有益的，但是我们希望我们不希望以后执行它。注释和 pass 的区别在于，注释完全被 Python 解释器忽略，而 pass语句没有被忽略。
